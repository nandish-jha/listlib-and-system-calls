Part A3 Comprehensive Test Plan
=======================================

Overview:
---------
This test plan provides exhaustive testing coverage for Part A3, which
implements a multi-threaded solution using POSIX pthreads (pthread library)
to compute squares using a recursive algorithm. The implementation creates a
parent thread that spawns multiple child threads using pthread_create(), each
calling Square(N) for values 1 to N until a deadline is reached.

Implementation Details:
----------------------
- Uses POSIX pthreads library with pthread_create() for thread creation
- Parent thread created in main() function using pthread_create()
- Child threads created by parent thread using pthread_create()
- Global volatile flag 'keep_running' controls execution
- Thread-local storage for current_thread identification
- Timing measurements using gettimeofday() for microsecond precision
- Memory allocation for pthread_t handles, IDs, and square counters
- Thread termination using pthread_detach() for fire-and-forget behavior
- Uses sleep() for timing delays (POSIX sleep function)
