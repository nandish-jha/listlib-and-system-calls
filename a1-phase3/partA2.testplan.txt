Part A2 Comprehensive Test Plan
=======================================

Overview:
---------
This test plan provides exhaustive testing coverage for Part A2, which
implements a multi-threaded solution using UBC pthreads library to compute
squares using a recursive algorithm. The implementation creates a parent
thread that spawns multiple child threads using the Create() function, each
calling Square(N) for values 1 to N until a deadline is reached.

Implementation Details:
----------------------
- Uses UBC pthreads library with Create() function for thread creation
- Parent thread created in mainp() function using Create()
- Child threads created by parent thread using Create()
- Global volatile flag 'keep_running' controls execution
- Thread-local storage for current_thread identification
- Timing measurements using gettimeofday() for microsecond precision
- Memory allocation for thread PIDs, IDs, and square counters
- Thread termination using Kill() function
- Uses Sleep() for timing delays (UBC-specific sleep function)
